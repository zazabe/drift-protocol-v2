Fields§

§scaled_balance: u64

The scaled balance of the position. To get the token amount, multiply by the cumulative deposit/borrow interest of corresponding market. precision: SPOT_BALANCE_PRECISION

§open_bids: i64

How many spot bids the user has open precision: token mint precision

§open_asks: i64

How many spot asks the user has open precision: token mint precision

§cumulative_deposits: i64

The cumulative deposits/borrows a user has made into a market precision: token mint precision

§market_index: u16

The market index of the corresponding spot market

§balance_type: SpotBalanceType

Whether the position is deposit or borrow

§open_orders: u8

Number of open orders

§padding: [u8; 4]
